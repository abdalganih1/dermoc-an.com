# 🧠 Agent.md - Dermocean B2B Catalog

## 📊 نظرة عامة
- **نوع المشروع:** منصة عرض منتجات B2B (كتالوج احترافي)
- **اللغة:** HTML / CSS / JavaScript (Vanilla) + Python for Backend Scripts
- **الإطار:** Custom Vanilla JS (Multilingual SPA)
- **الإصدار:** v3.0 (3-Section Restructuring)
- **نقطة الدخول:** `index.html`

## 🌲 المخطط الشجري
project/
├── data/
│   ├── products.json        # قاعدة بيانات المنتجات الأساسية (21 منتج)
│   ├── catalog_extracted.json # البيانات المستخرجة من PDF
├── css/
│   ├── style.css            # التنسيقات الرئيسية (يتضمن تنسيقات 3-Sections الجديدة)
├── js/
│   ├── products.js          # متحكم عرض المنتجات (ProductLoader v3.0)
│   ├── utils.js             # وظائف عامة
├── scripts/ (Python Tools)
│   ├── extract_catalog_data.py # لاستخراج البيانات من PDF الكتالوج
│   ├── update_products.py      # لتحديث قاعدة البيانات بالمنتجات الجديدة
│   ├── generate_product_images.py # لتوليد صور المنتجات غير الموجودة (AI)
│   ├── deploy_cf_pages.py      # لرفع الموقع مباشرة إلى Cloudflare Workers (Assets)

## 🛠️ أوامر التشغيل
| الأمر | الوظيفة |
|-------|---------|
| `python update_products.py` | دمج وتحديث قاعدة بيانات المنتجات وإعادة الهيكلة للأقسام |
| `python -m http.server 8000`| تشغيل بيئة التطوير محلياً عبر منفذ 8000 |
| `python deploy_cf_pages.py --token <TOKEN> --account <ACCOUNT_ID>` | نشر الموقع مباشرة على Cloudflare |

## ✅ أفضل الممارسات المكتشفة
- **دعم اللغات (i18n):** إدارة النصوص يتم غالباً محلياً من خلال Data attributes في الواجهة و Objects داخل `products.js`.
- **المنتجات الذكية:** استخدام الصور بالذكاء الاصطناعي للمنتجات المفقودة (مثل منتجات الميزوثيرابي) للحفاظ على التصميم الموحد الاحترافي.
- **التخطيط (Layout):** تم الاعتماد على تصميم "3 أقسام" (Peeling, Skincare, Mesotherapy) كبديل للفلاتر القديمة لتسهيل العرض وتسويق كل فئة بشكل واضح.
